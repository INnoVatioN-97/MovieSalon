## Introduction

#### 1. 개요

Movie Salon은 TMDB API와 KOBIS(영화진흥위) API를 사용하여 제작한 반응형 웹 애플리케이션 입니다.

- 국내 박스오피스 순위 및 영화 정보 확인 가능 [KOBIS API & TMDB API ]
- 글로벌 박스오피스 리스트 및 개봉예정 영화 목록 확인 가능 [TMDB API]
- 영화 검색 기능을 통해 찾고자 하는 영화 정보 확인 가능 [TMDB API]
- 인물의 필모그래피 확인 가능 [TMDB API]
- 글로벌 박스오피스 페이지서 영화 프리뷰 다이얼로그 구현(포스터, 줄거리, 예고편)[TMDB API]
- 영화 정보 페이지서 [한줄평 기능](https://user-images.githubusercontent.com/68140483/138848612-86780211-2ed5-45f6-8191-a1391213e65f.png) 구현 [FireBase DB]
- 로그인 기능 Social Login 적용 [ Google&github ]
